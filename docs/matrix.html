gauss_elim(self, second: Union[ForwardRef('Matrix'), NoneType] = None, step_by_step: bool = False) → 'Matrix'¶

Calculate with gauss elimination. Returns right-side matrix after calculation.

Parameters
second

Right-side matrix. can be 1D or 2D. If not given, this method calculates inverse matrix.

step_by_step

If true, print step by step solution.

Returns
Matrix

Result of gauss elimination calculation.

Raises
ValueError

If self is not square matrix, or length is different.

ZeroDivisionError

If cannot calculate due to divide with zero. It happens when determinent of self is zero.

Examples

Example of using gauss_elem with no option:

>>> a = Matrix([[1,2,3],[2,5,3],[1,0,8]])
>>> print(a.gauss_elem())
[[-40 16  9]
 [ 13 -5 -3]
 [  5 -2 -1]]

Example of using gauss_elem with option:

>>> b = Matrix([[1,0,0],[0,1,0],[0,1,1]])
>>> print(a.gauss_elem(b))
[[-40 16  9]
 [ 13 -5 -3]
 [  5 -2 -1]]

Example of using step by step solution:

>>> a.gauss_elem(step_by_step=True)
1 2 3 | 1 0 0
2 5 3 | 0 1 0
1 0 8 | 0 0 1
Add row 1 * -2 to row 2
